MOONDUST MINER

MoonDust Miner is an idle game written in Solidity. In the game, everyone has a MoonBase where MoonRovers collect MoonDust to mine for Unobtainium. Players can choose to buy MoonRovers, compound the MoonDust into Unobtainium or it can be sold for a reward in BNB. The game is a shameless copy of the work pioneered by Ether Shrimp Farm.

- Base daily returns 4%

- Fee 6%

- Referral 5%
